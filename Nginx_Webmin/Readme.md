### Install Nginx Module in Webmin

To easily set up and manage Nginx through Webmin, follow these steps:

1. **Access Webmin Panel**  
   Log into your Webmin panel.

2. **Install the Nginx Module**  
   - Navigate to **Webmin > Webmin Configuration > Standard Modules**.
   - In the "Available modules" section, search for **Nginx**.
   - Click **Install** to add the Nginx module to your Webmin panel.

3. **Configure Nginx through Webmin**  
   After installation, you'll find an option to configure Nginx directly from the Webmin interface, making it easier to manage your server settings.

This module allows you to:
- Manage Nginx configuration files.
- Restart/reload Nginx.
- Set up virtual hosts, SSL, and other configurations.
