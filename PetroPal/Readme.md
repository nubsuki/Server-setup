## My Backup Bot to Help Back Up Stuff

[PteroPal](https://github.com/nubsuki/PteroPal)

### Features:

- Integrates with game server management tools like Pterodactyl.
- Includes options for cloud storage backups (e.g., Google Drive).

Find more details and setup instructions on the [PteroPal GitHub page](https://github.com/nubsuki/PteroPal).
